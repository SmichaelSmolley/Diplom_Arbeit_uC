# Hardware config
Bitte jeweils einen eigenen Branch für jede Hardware erstellen (in VS Code).

Wenn nicht in VS code erstellt (mit PlatformIO) dann einfach ordner und die Arduino IDE's einfügen.

# Arduino IDE

In der Arduino IDE unter **Preferences → Additional Boards Manager URLs** folgende URL eintragen:

https://www.pjrc.com/teensy/package_teensy_index.json

Danach das **Teensy Board Package** installieren.
